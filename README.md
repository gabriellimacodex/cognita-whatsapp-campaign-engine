# Cognita WhatsApp Campaign Engine

Product and technical planning for a WhatsApp campaign engine with:

- scheduled group communications via UAZAPI;
- official 1:1 opt-in flow via Kapso/Capsule and Meta WhatsApp API;
- extracted group contacts stored as discovered contacts;
- approval-first campaign workflow;
- agentic chat with image and audio instructions;
- Apple-grade operational frontend direction.
